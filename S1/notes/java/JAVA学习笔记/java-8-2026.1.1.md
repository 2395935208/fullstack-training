# java学习第八天
## default的位置和省略
- default和case的位置可以任意书写，为了方便阅读一般是case从小到大写，default在最下面
- defaault可以省略不写，如果所有的case都不匹配，则没有任何输出结果
## case穿透现象
如果与case值匹配后，没有break，就会继续执行下一个case
## switch新特性
**yield 关键字**
![yield](<屏幕截图 2026-01-01 120244.png>)
case->表示，后面加大括号就不用写break了
其中大括号里写yield可以直接写结果，然后把switch赋值给一个变量，最后可以直接打印这个变量
## if和switch该怎么选
用于判断，并且是范围的时候用if
对于有限个的选择用switch
## 循环语句
- for循环
三要素：开始条件，结束条件，循环条件
每次写之前明确**开始条件**，**结束条件**，**循环体**都是什么
## while循环和for循环的区别
for循环适合知道范围，while适合知道结束条件的
