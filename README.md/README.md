# Fullstack Learning Roadmap

这是我的 **6个月全栈开发训练计划仓库**。
目标是通过持续项目实践，掌握 **Java后端 + 前端开发 + AI应用开发**，并达到可投实习/校招的技术水平。

# 学习目标

* 掌握 Java 后端开发（Spring Boot）
* 具备前端开发能力（HTML / CSS / JavaScript / React）
* 能开发完整前后端系统
* 掌握基础 AI 应用开发（FastAPI + AI API）
* 完成多个可写入简历的项目
